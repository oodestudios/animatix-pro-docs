# Text Animations

**Text animations** control letter-by-letter and word-by-word reveals, ideal for cinematic titles and dialogue effects.

## Overview

**Purpose:** Create dynamic text effects and reveals

**Common Use Cases:**

- **Typewriter Effects:** Character-by-character typing
- **Text Reveals:** Word-by-word or line-by-line reveals
- **Character Effects:** Individual letter animations
- **Text Effects:** Rainbow colors, flickering, blinking
- **Special Effects:** Explosion, gravity drop, curtain reveal

## Animation Types

### Reveal Animations

- **Typewriter** — Character-by-character typing
- **Scramble Text** — Text scrambling effect
- **Word Fade In** — Word-by-word fade reveal
- **Line Drop** — Line-by-line drop animation
- **Fade Up** — Text fade with upward movement
- **Grow and Fade** — Scale with fade effect

### Character Effects

- **Character Scale Pop** — Bouncy character scale
- **Character Fade** — Character-by-character fade
- **Character Slide In** — Character slide reveal
- **Character Bounce** — Character drop bounce
- **Character Shake** — Character shake effect
- **Character Jitter** — Character jitter motion
- **Character Wave** — Character wave motion
- **Character Flip Reveal** — Character flip reveal
- **Character Rotate Reveal** — Character rotate reveal
- **Character Zoom In** — Character zoom effect

### Text Effects

- **Rainbow Text Color** — Rainbow color cycling
- **Text Flicker** — Text flickering effect
- **Blinking** — Blinking text animation

### Special Effects

- **Explosion Reveal** — Character explosion effect
- **Gravity Drop** — Gravity-based drop
- **Curtain Reveal** — Curtain-opening reveal
- **Crumble Reveal** — Crumbling text reveal

## Common Parameters

- **Duration:** 0.8–3.0s typical (character effects: 0.5–1.5s, reveals: 1.0–5.0s)
- **Easing:** EaseOutQuart for smooth reveals, EaseOutBack for bouncy effects
- **Loop Type:** None (one-time reveals), Loop for continuous effects (Rainbow, Blinking)

## Best Practices

- Use typewriter effects for dialogue and narrative text
- Use character effects for attention-grabbing titles
- Use reveal animations for cinematic text presentations
- Test text animations with different font sizes and styles
