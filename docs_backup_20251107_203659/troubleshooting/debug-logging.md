# Debug Mode and Logging

**Debug Mode and Logging** helps diagnose animation issues.

## Overview

**Purpose:** Enable debugging and logging for troubleshooting

**Key Features:**

- **Debug Logging:** Enable detailed logging
- **Performance Logging:** Log performance metrics
- **Error Logging:** Log errors and warnings
- **Event Logging:** Log event triggers

## Configuration

### GraphExecutor Debug Settings

- **Enable Performance Logging:** Log performance metrics
- **Enable Detailed Logging:** Log detailed execution information

### Event Handler Debug Settings

- **Enable Logging:** Enable event handler logging
- **Log Level:** Set log level (Info, Warning, Error)

## Best Practices

- **Enable logging for debugging:** Use logging to diagnose issues
- **Disable logging in production:** Disable logging for release builds
- **Use appropriate log levels:** Use appropriate log levels
- **Review logs regularly:** Review logs to identify issues
