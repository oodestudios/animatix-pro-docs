# Performance Optimization

**Performance Optimization** ensures animations run smoothly on all platforms.

## Overview

**Purpose:** Optimize animations for performance

**Key Strategies:**

- **Reduce node count:** Minimize unnecessary nodes
- **Optimize durations:** Use appropriate animation durations
- **Limit parallel execution:** Maximum 2 nodes in parallel
- **Mobile optimization:** Reduce complexity for mobile

## Best Practices

- **Minimize nodes:** Remove unnecessary nodes
- **Optimize durations:** Use shorter durations for quick feedback
- **Limit parallel:** Maximum 2 nodes in parallel
- **Test on target platforms:** Test on actual devices
- **Profile performance:** Use Unity Profiler
