# UIAnimationEventSystem Component

**UIAnimationEventSystem** handles UI-specific animation events and callbacks.

## Overview

**Purpose:** Handle UI-specific animation events

**Key Features:**

- **UI Events:** Handle UI animation events
- **Event Callbacks:** Handle UI event callbacks
- **Integration:** Bridge UI systems with animations

## Configuration

| **Property** | **Type** | **Description** |
|--------------|----------|-----------------|
| **Target Canvas** | `Canvas` | Target canvas for UI events |
| **Enable Logging** | `bool` | Enable debug logging |

## Best Practices

- Assign Target Canvas in Inspector
- Use UI event system for UI-specific animations
- Enable logging for debugging
- Test UI events in Play Mode
