# Platform-Specific Issues

**Platform-Specific Issues** address problems unique to different platforms.

## Overview

**Purpose:** Resolve platform-specific animation issues

**Common Platforms:**

- **Desktop:** Windows, Mac, Linux
- **Mobile:** iOS, Android
- **WebGL:** Web browsers
- **Console:** PlayStation, Xbox, Nintendo Switch

## Common Issues

### Mobile Issues

- **Performance:** Animations run slowly on mobile
- **Battery:** Animations drain battery
- **Memory:** Animations use too much memory

### WebGL Issues

- **Compatibility:** Animations don't work in browsers
- **Performance:** Animations run slowly in browsers
- **Limitations:** WebGL has limitations

## Solutions

### Mobile Solutions

- **Reduce complexity:** Simplify animations for mobile
- **Optimize particles:** Reduce particle counts
- **Test on devices:** Test on actual mobile devices

### WebGL Solutions

- **Check compatibility:** Verify browser compatibility
- **Optimize performance:** Optimize for WebGL
- **Test in browsers:** Test in actual browsers
