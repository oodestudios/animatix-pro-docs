# Common Issues and Fixes

**Common Issues** and their solutions for Animatix Pro.

## Overview

**Purpose:** Resolve common animation issues

**Common Issues:**

- **Graph not playing:** Graph doesn't execute
- **Node errors:** Nodes show errors
- **Missing connections:** Connections not working
- **Performance issues:** Animations run slowly

## Solutions

### Graph Not Playing

- **Check GraphExecutor:** Ensure GraphExecutor is enabled
- **Check Graph Asset:** Verify GraphFlow Asset is assigned
- **Check Auto Start:** Verify Auto Start is enabled if needed
- **Check Connections:** Ensure nodes are connected properly

### Node Errors

- **Check node configuration:** Verify node properties are set correctly
- **Check target references:** Ensure target objects are assigned
- **Check dependencies:** Verify required components exist

### Missing Connections

- **Check connection validity:** Verify connections are valid
- **Check node ports:** Ensure ports are compatible
- **Check graph structure:** Verify graph structure is correct
